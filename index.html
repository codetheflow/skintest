A few minutes for set up

  • 1Install git
  • 2Install NodeJS
  • 3Install npm
  • 4And just execute:
To get started with development:

  git clone https://github.com/codetheflow/skintest-starter.git
cd skintest-starter
npm i
  
  Copied!

In the first terminal:

  nmp run build:watch
  
  Copied!

In the second terminal:

  nmp run start:watch
  
  Copied!

Coding and test run example

Language that helps to develop

        
feature('todos add')

  .before('scenario'
    , I.open(page.start)
    , I.open(todos.url)
    , I.open(clear_todos)
  )

  .scenario(‘ check the list has all added items’
    , I.do(add_todo, ‘learn testing’)
    , I.do(add_todo, ‘be cool’)
    , I.test(‘list contains added items’)
    , I.see(todos.list, has.lenght, 2)
    , I.see(todos.item_label_at(0), has.text, ‘learn testing’)
    , I.see(todos.item_label_at(1), has.text, ‘be cool’)
  )
